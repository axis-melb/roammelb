📶 eSIM 基础
eSIM 是内置在手机里的虚拟SIM卡，无需插入实体卡。购买后即时发送二维码到邮箱，扫码安装，全程在线完成。与实体SIM最大的区别：无需到机场排队，无需登记身份证，不占用手机实体卡槽，可与澳洲号码同时使用。
RoamMelb 销售的是纯数据流量（Data-Only）eSIM，不含手机号码和短信（SMS）功能，因此无需实名登记。您仍然可以正常使用所有数据流量支持的语音服务，例如 WhatsApp、微信语音通话等。
主流支持机型：iPhone XR / XS 及以上 · Samsung Galaxy S20系列及以上 / Z Fold / Z Flip · Google Pixel 3 及以上 · OnePlus 11/12/13。

⚠️ 注意：在中国大陆、香港购买的手机（包括三星、小米、华为）很多不支持eSIM，即使同款机型的澳洲版本支持。检查方法：iPhone → 设置 → 通用 → 关于本机 → 确认有"可用SIM卡"；Android → 拨打 *#06# → 查看是否有EID号码。不确定请联系 contact@roammelb.com
RoamMelb 是 Airalo 官方授权的转售商（Reseller）。您购买的 eSIM 产品与直接在 Airalo 购买完全相同，由同一网络运营商提供，品质无异。RoamMelb 额外提供：中文客服支持、支付宝 / 微信支付等本地支付方式，以及中文出行资讯。
📲 安装与激活
付款后，eSIM 二维码即时发送到您的注册邮箱。

重要提示：扫描二维码的手机，必须是您要安装 eSIM 的那台手机。由于扫码和安装在同一台手机上无法同时进行，您需要用另一台设备（第二台手机、平板、电脑）打开邮件显示二维码，再用要安装的手机扫描。

iPhone：设置 → 蜂窝网络 → 添加eSIM → 扫描二维码
Android（Samsung）：设置 → 连接 → SIM卡管理器 → 添加eSIM
Android（Pixel）：设置 → 网络和互联网 → SIM卡 → 下载SIM

安装后，请将 eSIM 设为数据SIM并开启「数据漫游」。
💡 建议在出发前在家 Wi-Fi 环境下完成安装，到目的地后再开启数据，省去机场操作的麻烦。
eSIM 从首次连接到目的地网络时才开始计时，不是从购买时开始。您可以提前几天买好并安装，到达目的地后再打开数据连接，有效期才正式启动。推荐出发前在家安装好，关闭 eSIM 数据，落地后再开启。
📊 使用与流量
微信/WhatsApp 文字消息：数天 · 语音通话（约40MB/小时）：约25小时 · 视频通话（约300MB/小时）：约3小时 · 地图导航：约14小时 · 社交媒体浏览（无视频）：约7小时 · YouTube 480p：约2小时。

省流量小技巧：出发前在 Wi-Fi 下下载离线地图；关闭 App 后台数据刷新；YouTube 设为 480p。
Airalo 无限量套餐实际上是：每日 3GB 高速流量，用完后自动降速至约 1Mbps（低速）。降速后微信文字、邮件、导航仍可正常使用；视频通话和高清视频会卡顿。大多数旅行者每日 3GB 已经足够。
大多数套餐支持开启个人热点，可与平板、笔记本共享网络。热点共享会加快流量消耗，有限额套餐请谨慎使用。
可以通过 Airalo App（App Store / Google Play 搜索「Airalo」）查看剩余流量并在线充值（Top-Up），无需重新安装 eSIM。
